# Usage
<!-- usage -->
```sh-session
$ npm install -g @omen/cli
$ omen COMMAND
running command...
$ omen (-v|--version|version)
@omen/cli/0.1.3 win32-x64 node-v8.9.3
$ omen --help [COMMAND]
USAGE
  $ omen COMMAND
...
```

## `omen help [COMMAND]`

display help for omen

```
USAGE
  $ omen help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
